03-2 링크 만들기

프론트엔드 개발 배우기
    href : 링크한 문서나 사이트의 주소
    target : 링크한 내용이 표시될 위치(현재 창 또는 새 창)를 지정
    download : 링크한 내용을 다운로드
    rel : 현재문서와 링크한 문서의 관계
    hreflang : 링크한 문서의 언어를 지정
    type : 링크한 문서의 파일유형 알려줌
프론트엔드 개발 배우기 target="_blank" => 새창/새탭에서 뜬다.
프론트엔드 개발 배우기 target="_self" => 현재 화면에서 뜬다.
